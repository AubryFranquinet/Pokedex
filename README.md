
# Pokedex

Pure JS project made with PokéAPI to make an online pokédex.


## API Reference

#### Get all pokemons

```http
  https://pokeapi.co/api/v2/[insert pokemon name]

```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `id` | `string` | get pokemon ID |
